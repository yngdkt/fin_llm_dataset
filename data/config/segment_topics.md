## 利用方法

### Python での読み込み例

```python
import json

# JSONL ファイルの読み込み
topics = []
with open('data/config/segment_topics.jsonl', 'r', encoding='utf-8') as f:
    for line in f:
        if line.strip():
            topics.append(json.loads(line))

# セグメント別にグループ化
from collections import defaultdict
by_segment = defaultdict(list)
for topic in topics:
    by_segment[topic['segment']].append(topic)

# 検索キーワードの取得
for topic in topics:
    ja_keywords = topic['keywords_ja']
    en_keywords = topic['keywords_en']
    # ... 検索処理
```

### 検索用クエリの生成例

```python
def generate_search_queries(topic: dict, language: str = 'ja') -> list:
    """トピックから検索クエリを生成"""
    if language == 'ja':
        return topic['keywords_ja']
    else:
        return topic['keywords_en']

# 使用例
for topic in topics:
    if topic['segment'] == 'Securities & Investment Banking':
        queries = generate_search_queries(topic, 'en')
        for query in queries[:5]:
            print(f"Searching: {query}")
```
